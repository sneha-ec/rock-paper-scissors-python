# Rock Paper Scissors Game 🎮

A simple Rock Paper Scissors game developed in Python. The user plays against the computer through the command line.

## Features
- User vs Computer gameplay
- Random computer choice using Python's `random` module
- Input validation
- Displays the winner or draw

## Technologies Used
- Python 3
- random module

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/sneha-ec/rock-paper-scissors-python.git
   ```

2. Open the project folder.

3. Run the program:
   ```bash
   python rock_paper_scissor.py
   ```

## Sample

```
rock, paper or scissor? rock
Computer chose: paper
Computer wins!
```
